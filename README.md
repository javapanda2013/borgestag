# BorgesTag

Firefox 拡張機能 — 右クリックまたはホバーボタンから画像をタグ・権利者付きで保存し、保存先フォルダを自動振り分けする拡張機能です。

---

## 動作環境

| 項目 | 要件 |
|------|------|
| ブラウザ | Mozilla Firefox（最新版推奨） |
| OS | Windows 10 / 11 |
| Python | 3.8 以上（パスが通っていること） |
| Python ライブラリ | Pillow（`install.bat` で自動インストール） |

> Python がインストールされていない場合は [python.org](https://www.python.org/downloads/) からダウンロードしてください。インストール時に **「Add Python to PATH」にチェック** を入れてください。

---

## 導入手順

### 1. ファイルの準備

1. 配布された ZIP ファイル（`image_saver_with_tags_x.x.x.zip`）を任意のフォルダに展開します。
2. 展開後のフォルダに `manifest.json`、`src/`、`native/` が存在することを確認してください。

> ⚠️ 展開先のフォルダは **移動・削除しないでください**。ネイティブアプリがこのパスを参照します。

---

### 2. ネイティブアプリの設定

1. `native/` フォルダを開きます。
2. **`install.bat`** をダブルクリックして実行します。
3. コマンドプロンプトが開き、以下の処理が自動実行されます。
   - Pillow（画像処理ライブラリ）のインストール
   - Windows レジストリへのネイティブメッセージングホストの登録
4. 「完了しました」のメッセージが表示されたらウィンドウを閉じます。

> 管理者権限を求めるダイアログが表示された場合は「はい」を選択してください。

> ✅ `native/image_saver.py` を更新した場合は `install.bat` の再実行が必要です。拡張機能（`manifest.json`）のみの更新では再実行不要です。

> ⚠️ フォルダを移動した場合は `install.bat` を再実行してください。

---

### 3. Firefox への拡張機能のインストール

1. Firefox を起動し、アドレスバーに `about:debugging` と入力して Enter を押します。
2. 左メニューの **「この Firefox」** をクリックします。
3. **「一時的なアドオンを読み込む...」** ボタンをクリックします。
4. 展開したフォルダ内の **`manifest.json`** を選択して開きます。
5. 「BorgesTag」が拡張機能の一覧に表示されれば完了です。

> 「一時的なアドオン」としてのインストールのため、**Firefox を再起動すると拡張機能がリセット**されます。再起動後は手順 3〜5 を繰り返してください。データは**エクスポート/インポート**機能で保持できます。

---

### 4. 動作確認

1. 任意の Web ページで画像の上で**右クリック**、または画像にマウスを乗せて表示される **💾 保存ボタン**をクリックします。
2. コンテキストメニューに **「画像をタグ付きで保存」** が表示されることを確認します。（右クリックの場合）
3. クリックすると保存ウィンドウが開きます。
4. 保存先フォルダを選択し、「保存」ボタンを押してファイルが保存されることを確認します。

> ⚠️ 保存ウィンドウが開かない場合や「Native Messaging エラー」が表示される場合は、`install.bat` を再実行してください。

---

### 5. 初期設定（推奨）

初回インストール後、設定画面（拡張機能アイコンを右クリック →「拡張機能のオプション」）から以下の設定を行うことを推奨します。

**保存先フォルダの初期設定**

1. 設定画面の **「全般」タブ**を開きます。
2. 「エクスプローラーの初期フォルダ」欄にルートフォルダのパスを入力し、「設定」ボタンを押します。
   例: `C:\Users\username\Pictures`

**エクスポート先の設定（データ保護）**

1. 設定画面の **「全般」タブ**を開きます。
2. 「標準エクスポート先フォルダ」に任意のパスを設定します。
3. 「即エクスポート保存」を ON にすると、エクスポート時に自動でファイルが保存されます。

> ✅ 設定完了後はエクスポートしてバックアップを取ることを推奨します。
