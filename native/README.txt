====================================================
  Image Saver with Tags — Native アプリ 更新手順
====================================================

■ 更新時の注意
  image_saver.py を更新した場合は、必ず install.bat を再実行してください。
  再実行しないと、古いバージョンのまま動作し続けます。

■ install.bat の実行手順
  1. このフォルダ内の install.bat をダブルクリック
  2. 管理者権限の確認ダイアログが出た場合は「はい」を選択
  3. 「完了しました」のメッセージを確認してウィンドウを閉じる
  4. Firefox を再起動する（再読み込みの場合は拡張機能を再インストール）

■ install.bat が行う処理
  - Pillow（画像処理ライブラリ）のインストール
  - Windows レジストリへの Native Messaging ホストの登録
    （拡張機能と image_saver.py を紐付けるための設定）

■ バージョン確認
  image_saver.py の先頭付近に「version: x.x.x」と記載されています。
  設定画面の「全般」タブのバージョン表示と合わせて確認してください。

■ ファイルを移動・削除しないでください
  install.bat はこのフォルダの絶対パスをレジストリに登録します。
  フォルダを移動した場合は install.bat を再実行してください。
